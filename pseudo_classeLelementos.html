Trabalhando com Pseudo-clase

São usadas para definir um elemento com um estado especial

Serve para:

  • Estilizar um elemento quando um usuário parra o mouse sobre ele.
  • Estilidar o link visitado e não visitado de forma diferente
  • Estilizar um elemento quando ele recebe foco

PSEUDO-CLASSE

Pseudo Elemento

Um Pseudo-elemento é usado para estilizar partes especificas de um elemento.

Por exemplo:

  • Estilize a primeira letra ou linha de um elemnto.
  • Insira conteúdo antes ou depois do conteúdo de um elemento.

>NOTA: O ::first-line só pode ser aplicado a um elemento de um nivel de bloco.

As seguinte propriedades se aplicam so ::first-line:

  • Propriedades da fonte
  • Propriedades de cor
  • Propreidades de fundo
  • Espaçamento entre as palavras
  • Espaçamentos entre letras
  • Decoração do texto
  • Alinhamento vertical
  • Transformação do texto
  • Altura da linha
